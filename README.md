## DETEKSI HATE SPEECH BERBASIS MACHINE LEARNING MENGGUNAKAN LINEAR SUPPORT VECTOR MACHINE

Ini adalah proyek klasifikasi teks untuk mendeteksi ujaran kebencian pada data teks berbahasa Indonesia. Proyek ini dibuat untuk memenuhi tugas UTS mata kuliah Kecerdasan Buatan. Model yang digunakan adalah Linear Support Vector Machine dengan metode TF IDF sebagai representasi fitur. Hasil evaluasi menunjukkan nilai akurasi sebesar 0.84 sehingga model dapat membedakan teks normal dan hate speech dengan cukup baik.
